# Week 3 - Advanced Security and Final Reporting

## Project
Securing the NodeGoat Application

## Objective
The objective of Week 3 was to perform advanced security testing, implement application logging, and create a security checklist to improve the security posture of the NodeGoat application.

## Tasks Performed

### 1. Basic Penetration Testing
- Performed Nmap port scanning.
- Conducted Stored XSS testing.
- Tested SQL Injection payloads.

### 2. Logging Implementation
- Installed Winston logging library.
- Implemented application startup logging.
- Generated security.log file.

### 3. Security Checklist
- Created SECURITY_CHECKLIST.md containing security best practices.
